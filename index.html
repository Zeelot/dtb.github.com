Hi! I'm David. I go by notJim on the internet.

This is my place where I put random stuff I'm playing around with. It turns out that pushing to a git remote is a lot easier than SFTPing into my VPN.

You may have guessed that I'm a web developer. In the past few years, I've written MVC applications in PHP and now in C#. Go ahead and ask me which I prefer.

I'm currently very interested in interaction design and UX, as well as API design and security.

Recently

I'm currently working at ThinkEco on our web-powered energy-efficiency product (an outlet that helps you save power). If you've never written a web app that interfaces with hardware, you're missing out.

At ThinkEco, my work involves designing user-friendly, gracefully-degrading user interfaces, as well as our REST API.
